# vuesjs-project-dfsr

Front : Vue.js v.2
Style : Tailwind v.3
Back : Node.js & Express.js
BDD : MongoDB & Mongoose

1. npm install
2. npm start
3. npm run serve

Si erreur UnhandledPromiseRejectionWarning : 
cmd terminal : npm run start --unhandled-rejections=strict ou/et npm run serve --unhandled-rejections=strict
