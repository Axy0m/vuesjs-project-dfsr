# vuesjs-project-dfsr

1. Front : Vue.js v.2
2. Style : Tailwind v.3
3. Back : Node.js & Express.js
4. BDD : MongoDB & Mongoose

**********************************

1. npm install
2. npm start
3. npm run serve

Si erreur UnhandledPromiseRejectionWarning : 
npm run start --unhandled-rejections=strict ou/et npm run serve --unhandled-rejections=strict
